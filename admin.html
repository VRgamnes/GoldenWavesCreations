🤖 GitHub Action Setup (Auto-Sync with Etsy)

Your site automatically syncs with Etsy every 6 hours using a GitHub Action. To set it up:

  1. Go to your GitHub repo → Settings → Secrets and variables → Actions
  2. Add secret ETSY_API_KEY — get this from Etsy Developer Portal ↗
  3. Add secret ETSY_SHOP_ID — find this in your Etsy shop URL or settings
  4. Go to Actions → Fetch Etsy Products → Run workflow to sync now

✨ Every time a price changes, you add a new shirt, or Printify updates something on Etsy, the GitHub Action will automatically update your website within 6 hours — or you can trigger it manually anytime.
